# Bella Napoli – Italian Restaurant Chatbot

Socket.IO chatbot for **Bella Napoli** with waiter bot **Luca**.

## Structure

- `client/` – React + Bootstrap 5 frontend (Vite)
- `server/` – Node.js + Express + Socket.IO backend

## Quick start

### Backend

```bash
cd server
npm install
npm start
```

Server runs on **http://localhost:3000**

### Frontend

```bash
cd client
npm install
npm run dev
```

Open **http://localhost:5173**

## Build frontend for production

```bash
cd client
npm run build
```

See `client/README.md` for component and Socket.IO event details.
